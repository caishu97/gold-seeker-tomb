# 摸金探险 - 3D古墓探险游戏

## 项目概述
一个基于 Web 浏览器运行的 3D 摸金探险游戏，玩家选择角色进入古墓探索，寻找宝箱、对抗怪物，在撤离点成功撤离将获得宝物。

## 技术栈
- **3D引擎**: Three.js (WebGL)
- **构建工具**: Vite (快速开发服务器与构建)
- **语言**: TypeScript
- **物理碰撞**: 自定义 AABB / 简单几何碰撞检测
- **地图生成**: 程序化古墓地图生成 (PCG - Procedural Content Generation)

## 核心玩法循环
1. **准备阶段**: 选择角色 (不同技能) → 用仓库的钱购买初始装备 → 确认进入古墓
2. **探索阶段**: 在古墓中搜索宝箱 → 击败怪物 → 收集宝物和武器
3. **撤离阶段**: 到达撤离点 → 成功撤离则所有物品回仓库 → 死亡则丢失本局所有装备
4. **经营阶段**: 仓库中售卖宝物获取金钱 → 购买更强的武器和辅助装备 → 进入下一轮

## 项目结构
```
├── src/
│   ├── core/           # 核心引擎 (渲染、场景管理、资源加载)
│   ├── game/           # 游戏逻辑 (状态管理、回合、胜负)
│   ├── map/            # 地图生成相关
│   ├── entities/       # 实体 (玩家、怪物、宝箱、武器)
│   ├── systems/        # 游戏系统 (战斗、经济、背包、AI)
│   ├── ui/             # UI界面 (HUD、菜单、背包、商店)
│   ├── utils/          # 工具函数
│   └── main.ts         # 入口文件
├── assets/             # 静态资源 (3D模型、贴图、音效)
├── index.html
└── package.json
```

## 开发约定
- 使用ES Modules模块化导入
- Three.js对象使用dispose()管理内存
- 游戏状态使用单一数据源 (GameState) 模式
- 坐标系: Y轴向上 (Three.js标准)
- 单位: 1 unit = 1 米

## 角色设计 (初版)
- **摸金校尉**: 移动速度快，对宝箱感知范围大
- **发丘中郎**: 防御高，武器伤害加成
- **搬山道人**: 对怪物伤害高，暴击率高
- **卸岭力士**: 负重高，血厚

## 地图设计
- 采用基于网格的古墓房间 + 走廊生成
- 随机确定：玩家出生点、撤离点、宝箱刷新点、怪物刷新点
- 3D立体结构：含多层结构，楼梯/斜坡连接

## 资源来源
- 使用程序化几何体 + 开源免费3D模型资源
- 后续可接入 kenney.nl 等免费资源包

## 运行方式

```bash
npm install
npm run dev    # 启动开发服务器
npm run build  # 构建生产环境
```
