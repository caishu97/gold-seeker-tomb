# 摸金探险 - 3D古墓探险游戏

> 这是xiaoyi尝试做的第二个游戏

一个基于 Web 浏览器运行的 3D 摸金探险游戏，玩家选择角色进入古墓探索，寻找宝箱、对抗怪物，在撤离点成功撤离将获得宝物。

## 核心玩法

1. **准备阶段**: 选择角色（摸金校尉、发丘中郎、搬山道人、卸岭力士）→ 购买装备 → 进入古墓
2. **探索阶段**: 在古墓中搜索宝箱 → 击败怪物 → 收集宝物和武器
3. **撤离阶段**: 到达撤离点 → 成功撤离则所有物品回到仓库 → 死亡则丢失本局装备
4. **经营阶段**: 仓库中售卖宝物获取金钱 → 购买更强的武器和辅助装备 → 进入下一轮

## 角色介绍

| 角色 | 特点 | 生命值 | 负重上限 |
|------|------|--------|----------|
| 摸金校尉 | 移动速度快，对宝箱感知范围大 | 100 | 45kg |
| 发丘中郎 | 防御高，武器伤害加成 | 130 | 50kg |
| 搬山道人 | 对怪物伤害高，暴击率高 | 90 | 40kg |
| 卸岭力士 | 负重高，血厚 | 150 | 70kg |

## 操作说明

- **W/A/S/D** - 移动
- **鼠标移动** - 控制视角
- **鼠标左键** - 攻击
- **Shift** - 加速跑（消耗体力）
- **走到绿色光圈** - 成功撤离

## 技术栈

- **3D引擎**: Three.js (WebGL)
- **地图生成**: 程序化古墓地图生成 (PCG)
- **语言**: JavaScript (ES6+ Modules)
- **构建工具**: Vite

## 本地运行

```bash
npm install
npm run dev    # 启动开发服务器
```

或者直接使用静态服务器：
```bash
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

## 项目结构

```
src/
├── core/           # 核心引擎 (渲染、场景管理)
├── game/           # 游戏逻辑 (状态管理)
├── map/            # 地图生成相关
├── entities/       # 实体 (玩家、怪物、宝箱)
├── systems/        # 游戏系统 (战斗)
└── ui/             # UI界面
```
